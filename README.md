# SwapShelf frontend

React + Vite app for the Panini World Cup 2026 sticker swap platform.

## Local development

```
npm install
npm run dev
```

## Deploying to Vercel

1. Push this repo to GitHub.
2. On vercel.com, import the repo.
3. Vercel auto-detects Vite — no config needed.
4. Deploy.

The backend API URL is currently hardcoded in `src/App.jsx` 
(the `API_BASE` constant at the top). Update it there if your 
backend URL changes.
